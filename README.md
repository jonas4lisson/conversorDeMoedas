# Conversor de moedas

![GitHub Org's stars](https://img.shields.io/github/stars/jonas4lisson?style=social)

## Descrição do projeto
Esse projeto tem como objetivo fazer o usuário poder realizar consultas reais de valores em qualquer moeda de países estrangeiros.
Compara-se os valores e um histórico de todas as moedas convertidas numa lista.

## ✔️ Técnicas e tecnologias utilizadas

- ``Java 22``
- ``InteliJ IDEA``
- ``Biblioteca Gson``
- ``exchangerate-api``
